# Servidor para o site "Steph Hoel Blog"

## Post

| Método | Caminho              |
| ------ | -------------------- |
| Get    | `/post`              |
| Get    | `/post/:idPost`      |
| Get    | `/post/state/:state` |
| Post   | `/post`              |
| Put    | `/post`              |
| Delete | `/post/:idPost`      |

| Post   | `/forgot-pass`       |
| Post   | `/reset-pass/:token` |

| Get    | `/login/:idLogin`    |
| Delete | `/login`             |
