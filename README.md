# Servidor para o site "Steph Hoel Blog"

## Post

| Método | Caminho           |
| ------ | ----------------- |
| Get    | `/post`           |
| Get    | `/post/:idPost`   |
| Post   | `/post`           |
| Put    | `/post`           |
| Delete | `/post/:idPost`   |

## Login

| Post   | `/register`       |
| Post   | `/login`          |
| Put    | `/login`          |
| Get    | `/login/:idLogin` |
| Delete | `/login`          |
